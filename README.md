# mahjong

A library of Japanese Mahjong for cli-game, server, etc.
It may support other Mahjong at future.

## Command

- init 初始化/初期化
- start 开始/開始
  - shuffle 洗牌/シャッフル
  - stack 排列/並ぶ
  - haipai 发牌/配牌
- draw 摸牌/ドロー
- dahai 打牌/だはい
- chii 吃/チー
- pon 碰/ポン
- kan 杠/カン
  - Kan 明杠/明槓
  - AnKan 暗杠/暗槓
  - KaKan 加杠/加槓
- riichi 立直/リーチ
- ron 荣和/ロン
- tsumo 自摸/ツモ
- restart 流局/流局
